    • 2.3 Use HTML5 para crear un documento que contenga el siguiente texto: ...
    Use h1 para el título (la primera línea de texto), p para el texto (la segunda y tercer línea de texto). Inserte una regla horizontal entre el elemento h1 y el elemento p. Abra su nuevo documento en un navegador Web para ver el documento marcado.
